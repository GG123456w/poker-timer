# 扑克盲注计时器 - 局域网方案

适用于：单店使用（投屏电脑 + 裁判手机/平板在同一 WiFi）。

**优势**：100% 稳定、不依赖公网、零延迟。

---

## 🚀 快速开始

### 第一步：双击启动

在 `c:\Users\Administrator\Desktop\poker-timer\` 目录下：
- **双击 `start-server.bat`**

脚本会自动：
1. ✅ 检查 Node.js
2. ✅ 安装依赖（首次）
3. ✅ 构建最新版本
4. ✅ 开放 Windows 防火墙 4173 端口
5. ✅ 显示本机 IP
6. ✅ 启动 HTTP 服务

### 第二步：看启动后的输出

```
========================================
   服务已启动！

   投屏电脑：http://localhost:4173
   局域网访问：http://192.168.1.100:4173   ← 记下这个 IP
   后台管理：http://192.168.1.100:4173/#/admin
========================================
```

### 第三步：手机/平板连接

确保投屏电脑和手机/平板**连接同一 WiFi**（不是访客网络）。

手机浏览器打开 `http://192.168.1.100:4173`

- **投屏页**：`http://192.168.1.100:4173`
- **后台页**：`http://192.168.1.100:4173/#/admin`（首次默认账号 `admin` / `admin888`）

---

## 📱 投屏电脑操作

1. **投屏电脑**（连接大屏/电视/投影仪）
2. 浏览器打开 `http://localhost:4173`
3. 按 **F11** 进入全屏
4. **保持浏览器开着**

## 📱 裁判手机/平板操作

1. 浏览器打开后台页 `http://192.168.1.100:4173/#/admin`
2. 登录（首次：`admin` / `admin888`）
3. **开始/暂停/重置/调整** 等操作
4. 投屏页会实时同步变化

## 🔧 常见问题

### Q1: 手机访问显示"无法访问此网站"

**A**: 90% 是这 3 个原因：

1. **手机和电脑不在同一 WiFi**
   - 让手机连和投屏电脑**同一个 WiFi**（不是访客网络）

2. **电脑开了多个 WiFi（笔记本情况）**
   - 如果笔记本同时连了有线网 + WiFi，可能服务绑定到错误的网卡
   - 重启脚本，确认显示的 IP 是 WiFi 网卡的 IP

3. **Windows 防火墙阻挡**
   - 第一次启动会弹窗问"是否允许"，**必须点"允许"**
   - 如果错过了，手动运行（管理员 PowerShell）：
     ```powershell
     New-NetFirewallRule -DisplayName "PokerTimer-LAN" -Direction Inbound -LocalPort 4173 -Protocol TCP -Action Allow
     ```

### Q2: 怎么查电脑 IP？

A: 启动脚本会显示。或者手动查：
```powershell
ipconfig
# 找 "IPv4 Address"，一般是 192.168.x.x 或 10.x.x.x
```

### Q3: 手机能访问但加载很慢/样式错乱

A: 这是浏览器缓存问题。手机浏览器：
- **无痕模式**打开（推荐）→ Chrome/Edge: 设置 → 新建无痕标签页
- 或 **清除浏览器数据** → 重新访问

### Q4: 关掉脚本后手机访问不了

A: 这是正常的，**脚本是服务进程**。关掉 = 服务停止。

要让手机随时访问：
- 投屏电脑**不关机/不锁屏**
- 或用专门的"小服务器"（如旧手机、平板、NUC）

### Q5: 想换端口（如 8080）

A: 编辑 `start-server.bat`，把 `4173` 全局替换成 `8080`，同时更新防火墙规则：
```powershell
netsh advfirewall firewall delete rule name="PokerTimer-LAN"
netsh advfirewall firewall add rule name="PokerTimer-LAN" dir=in action=allow protocol=TCP localport=8080
```

### Q6: 异地访问（不在同一 WiFi）

A: 局域网方案**不支持异地访问**。如果需要：
- 方案 A：用 **Cloudflare Pages** 部署（公网 + 国内访问较好）
- 方案 C：买**自定义域名** + Cloudflare 加速

### Q7: 更新代码后怎么生效？

A: 关闭当前脚本 → 重新双击 `start-server.bat`（会自动 build 最新版本）。

## 🛑 停止服务

在脚本运行窗口按 **Ctrl + C**，或直接关闭窗口。

---

## 📋 系统要求

- **操作系统**：Windows 10/11
- **Node.js**：18+ （[下载](https://nodejs.org/)）
- **网络**：投屏电脑和手机/平板在同一 WiFi

---

## 💡 进阶：做成开机自启

把 `start-server.bat` 的快捷方式放到：
```
shell:startup
```
（按 Win+R，输入 `shell:startup`，回车）

这样电脑开机后自动启动服务。
